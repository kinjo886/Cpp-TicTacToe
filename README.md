# 井字棋 (Tic-Tac-Toe) - C++/EasyX

基于 C++ 和 EasyX 图形库开发的经典双人对战井字棋游戏。本项目展示了基础的图形绘制、鼠标交互处理以及游戏胜负逻辑判定。

## ✨ 功能特点

- **实时图形渲染**：利用 EasyX 的 `BeginBatchDraw` 和 `FlushBatchDraw` 实现双缓冲绘图，确保画面平滑无闪烁。
- **动态交互**：通过 `peekmessage` 实时捕获鼠标左键点击 (`WM_LBUTTONDOWN`)，实现精准落子。
- **状态提示**：游戏界面左上角实时显示当前轮到的棋子类型（'O' 或 'X'）。
- **智能判定**：
    - **胜负检测**：自动遍历横、竖、斜共 8 种获胜情况。
    - **平局检测**：当棋盘填满且无玩家获胜时自动判定为平局。
- **帧率控制**：代码中加入了 `GetTickCount` 计时逻辑，将游戏运行限制在 60 FPS，保证了性能的平稳。

## 🛠️ 技术细节

- **棋盘表示**：使用 `char board_data[3][3]` 二维数组存储棋盘状态（'-' 代表空，'O'/'X' 代表棋子）。
- **坐标映射**：将鼠标点击的像素坐标（如 600x600 窗口）映射到数组索引（0, 1, 2）。
- **图形绘制**：
    - 使用 `circle()` 绘制玩家 O。
    - 使用 `line()` 交叉绘制玩家 X。
    - 使用 `outtextxy()` 渲染 UI 文本信息。

## 🚀 环境准备

1. **开发环境**：Visual Studio 2022。
2. **图形库**：[EasyX Graphics Library](https://easyx.cn/)。
3. **项目配置**：
   - 包含目录指向 EasyX 的 `include` 文件夹。
   - 库目录指向 EasyX 的 `lib` 文件夹。
   - 建议在项目属性中将 **字符集** 设置为 `使用多字节字符集` 以配合代码中的文本处理。

## 🕹️ 运行说明

1. 编译并运行项目。
2. 鼠标左键点击棋盘空格进行落子。
3. 游戏结束后，会通过弹窗 (`MessageBox`) 提示获胜者或平局结果。
